# 说明

nodejs + express + mongodb + mongoose + es6/7 + vue + element-ui

# 项目运行

```
https://github.com/chenya1123236324/node-manage-system.git 

cd node-manage-system

npm install

开启 mongodb

npm run dev  访问: http://localhost:3000

```
